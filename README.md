# VSCode settings (recommended)
1. install plugins: nuxtr, volar and eslint
2. allow volar takeover mode (https://vuejs.org/guide/typescript/overview.html#volar-takeover-mode)


## Setup

Make sure to install the dependencies:

```bash
# bun
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# bun
bun run dev
```

## Production

Build the application for production:

```bash
# bun
bun run build
```

Locally preview production build:

```bash
# bun
bun run preview
```
